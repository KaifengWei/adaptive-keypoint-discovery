# 自适应关键点发现：重构路线

本目录是当前研究问题的唯一方案入口。研究只回答：

> 给定一幅秧苗图像，算法能否在没有人工关键点定义的情况下，根据当前植株的结构和形态，自主确定关键点的位置、数量及其稳定性？

当前状态：**V4整株数据集共300张；220张train已生成跨增强自动教师目标，核心DINOv2动态热图模型已在RTX 3090完成首轮80轮训练。40张val评估表明点集具有较高重复性，但代码审计确认现有路径主要由整幅骨架独立生成，学习点尚未真正决定拓扑。下一步推荐实现关键点条件结构图；表型准确率仍待人工参考，V4 test继续锁定。**

## 约束

- 不使用人工关键点坐标作为训练监督。
- 不预先规定每幅图必须输出相同数量的点。
- DINOv2首先作为冻结的独立视觉编码器验证，不从头训练。
- 第一阶段只做可行性试验；未通过门禁，不进入复杂模型训练。
- 暂不引入与核心发现问题无关的额外感知网络或弱监督扩展链路。
- 旧工程保留作数据与工具参考，但不再作为新路线的方法依据。
