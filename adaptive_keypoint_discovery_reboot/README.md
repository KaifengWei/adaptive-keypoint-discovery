# 自适应关键点发现：重构路线

本目录是当前研究问题的唯一方案入口。研究只回答：

> 给定一幅秧苗图像，算法能否在没有人工关键点定义的情况下，根据当前植株的结构和形态，自主确定关键点的位置、数量及其稳定性？

当前状态：**已从 `D:\kp\数据集图片` 建立独立的20张干净背景核心集，后续方法开发不再更换数据来源。G1′在6张 smoke 上获得1.000的白底前景命中率中位数、0.916的重复性F1中位数和约0.996的骨架覆盖率中位数。经研究目标修订，沿茎叶分布的较密点集可作为样条控制点或路径采样点，不再以“必须少量离散”作为门禁。G1′已通过候选阶段，下一步是完成点集排序、分支路径和表型效用验证；未启动深度训练。**

## 约束

- 不使用人工关键点坐标作为训练监督。
- 不预先规定每幅图必须输出相同数量的点。
- DINOv2首先作为冻结的独立视觉编码器验证，不从头训练。
- 第一阶段只做可行性试验；未通过门禁，不进入复杂模型训练。
- 暂不引入与核心发现问题无关的额外感知网络或弱监督扩展链路。
- 旧工程保留作数据与工具参考，但不再作为新路线的方法依据。
