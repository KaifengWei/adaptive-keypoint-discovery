# PROJECT STATE

更新时间：2026-07-22

## 一、已锁定的研究口径

唯一主线是：无人工关键点定义与坐标标签时，模型能否从单株秧苗图像中自适应发现数量可变、位置稳定、能支撑表型计算的关键点。

“发现成功”同时满足三层条件：

1. 点落在真实植株地上部结构上，并在合理图像增强下可重复；
2. 点能够连接、排序为正确的茎叶路径；
3. 由这些路径计算出的长度、曲率、叶倾角等表型接近人工测量或可靠参考。

## 二、当前有效结果

- G1 的单独 DINOv2 响应：重复性较好，但易集中于基部/主轴并漏掉叶尖。
- 局部对比峰和每图聚类中心：会产生较多无意义响应，不能单独作为关键点方案。
- G1′结构候选：在干净背景小样本上的前景命中和重复性较好，适合作为下一阶段候选生成基线。
- G1′当前仍是规则与表征融合的候选器，不是已学成的端到端模型；尚未证明路径连接与表型精度。
