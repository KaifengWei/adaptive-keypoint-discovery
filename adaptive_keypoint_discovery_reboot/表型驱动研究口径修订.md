# 表型驱动的自适应关键点研究口径修订

## 一、核心定义

本研究中的“自适应关键点发现”定义为：给定一幅秧苗图像，算法不依赖人工关键点坐标或固定点数规则，根据当前植株形态自动产生位置和数量可变的二维支撑点，并使该点集能够稳定重建秧苗路径、计算目标表型。

这里的“发现”不等于找到少量具有固定名称的器官点，也不要求每个点都具有独立的解剖学语义。它至少包含三层含义：

1. **位置由图像结构产生**：点位随茎叶形状、端点、分叉、弯曲和视觉特征变化；
2. **数量由当前实例和任务产生**：简单植株可以少，曲折或分支丰富的植株可以多；
3. **有效性由表型结果检验**：点集是否有用，取决于曲线、长度、宽度、角度、曲率及最终农艺评价是否准确稳定。

## 二、G1′的准确定位

G1′不是最终表型模型，而是当前最有希望的“自适应表型支撑点生成器”。它融合自动前景支撑、骨架端点/分叉、形状角点和冻结DINOv2证据，在干净背景smoke6上获得较高前景命中、重复性和骨架覆盖。

G1′产生较密的点并不构成方法错误。对于B样条拟合、叶片弧长、局部曲率和叶倾角计算，适当增加沿曲线分布的控制点通常比只有少量点更有利。需要实验决定的是“多到什么程度仍有收益”，而不是预先要求稀疏。

## 三、方法闭环

```text
RGB秧苗图像
  → 自动前景与骨架
  → 冻结DINOv2提供局部视觉证据
  → G1′产生可变数量支撑点
  → 骨架图完成分支划分与点序排序
  → 每条路径进行样条拟合
  → 结合前景掩膜计算表型
  → 形成单株表型表和健壮度评价输入
```

必须强调：关键点坐标本身不能直接给出全部表型。叶片或主轴长度、角度和曲率依赖有序路径；宽度和面积还依赖前景掩膜；叶片数、分蘖数等依赖拓扑；物理单位需要标尺或像素—毫米换算关系。

## 四、第一批目标表型

| 表型 | 主要输入 | 计算方式 | G1′点集作用 |
|---|---|---|---|
| 株高/纵向伸展 | 基部、最高点、前景 | 垂直距离或主路径投影 | 提供主轴和端点支撑 |
| 主轴/叶片长度 | 有序路径点 | 样条弧长积分 | 作为曲线控制或采样点 |
| 平均/局部宽度 | 掩膜、路径点 | 距离变换的两倍 | 确定测量中心位置 |
| 叶倾角 | 分支起点和路径切线 | 样条一阶导数夹角 | 支撑基部及局部切线拟合 |
| 弯曲度/曲率 | 有序路径点 | 样条二阶导数或弧弦比 | 密集点提高局部形态表达 |
| 投影面积 | 前景掩膜 | 前景像素总量 | 点集主要用于结构校验 |
| 端点/分支数量 | 骨架图 | 图拓扑统计与短枝剪除 | 点集辅助稳定拓扑节点 |

第一阶段先以图像像素和归一化单位完成算法闭环；只有存在标尺或可靠的像素—毫米换算后，才输出物理单位。

## 五、正确的消融问题

后续不再把“点越少越好”作为主张，而比较：

- 无DINOv2证据时，曲线和表型误差是否上升；
- 仅端点/分叉、加入形状角点、再加入DINOv2点时，表型精度如何变化；
- 不同最小间距或容量上限下，点数—表型误差曲线何时趋于饱和；
- 无几何稳定性筛选时，重复拍摄或扰动后的表型波动是否增大；
- 使用折线、B样条和平滑样条时，哪种对当前点集最稳定。

由此，G1′的价值最终由“同一植株扰动后表型是否稳定”和“自动表型与参考测量是否一致”决定，而不是由视觉上点数多少决定。
