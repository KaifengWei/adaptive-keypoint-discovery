## 无压缩复制示例

本机 PowerShell 中把目标地址替换成真实服务器：

```powershell
scp -r "D:\kp\adaptive_keypoint_discovery_reboot" user@server:/remote/project/path/
```

如果服务器已经能访问共享盘，直接在 VS Code Remote-SSH 中打开共享项目路径，无需再次复制。数据量大或网络不稳时优先使用 `rsync`，它能增量和断点续传。

远程 Codex 打开项目后第一句话可以只写：

> 请先完整读取 AGENTS.md、PROJECT_STATE.md 和 experiment/远程3090训练执行说明.md，然后从 remote_gpu_check.py 开始继续。

这样它不需要重新猜测研究边界或回顾全部聊天。
