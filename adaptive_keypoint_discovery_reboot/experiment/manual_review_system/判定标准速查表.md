# 人工复核判定标准速查表

## 平台一：地上部有效域

| 字段 | 通过 | 失败 | 不确定 |
|---|---|---|---|
| 地上部完整 | 所有可见茎叶均保留 | 叶尖、叶片或茎基部被截断 | 原图本身遮挡，无法确认 |
| 颖果根须排除 | 颖果、主根和根须不进入绿色有效域 | 根须或颖果仍被当作地上部 | 组织颜色与边界无法区分 |
| 基部过渡区 | 位于shoot侧、接近地上部起始位置 | 落在颖果、根须或明显远离基部 | 基部被遮挡或结构模糊 |

## 平台二：器官路径

| 字段 | 选择规则 |
|---|---|
| 路径语义 pass | 主体路径及必要分支均能对应真实茎叶，且不存在影响表型的明显错误 |
| 路径语义 fail | 存在漏叶、错连、错误基部或严重偏离，导致器官/表型解释不可靠 |
| 漏叶 yes | 原图存在清楚的叶片或短叶，但最终路径没有覆盖 |
| 错连 yes | 路径跨过白色背景、连接不同器官、进入根须，或曲线明显脱离真实叶片 |
| 基部 correct | 路径起点位于shoot侧地上部过渡位置 |
| 基部 wrong | 起点位于颖果、根须、叶片中段或明显偏离地上部起点 |

## 三条优先原则

1. 先看无标注原图，再看学习点结构图和最终路径；
2. “有输出”不等于“正确”，没有路径时不能因为没有错连而判整体通过；
3. 若选择`pass`，原则上漏叶和错连都应为`no`，否则必须在备注中解释。
