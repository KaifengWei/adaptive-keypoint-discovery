# 下一阶段改进与复核计划

## 当前证据

路线B将旧路线基部错误从28张降到4张、错连从11张降到5张，说明地上部有效域和shoot侧基部约束有效；但漏叶仍为8张，严格联合通过为22/40。

val-only阈值扫描表明，降低检测阈值主要增加冗余节点，不能系统补回小叶路径。因此下一轮不以“多取点”为主。

## 已确认的技术方向

下一轮采用“结构覆盖增强自动教师 + 局部尺度短枝解码”：

1. 自动识别通过持久性筛选的地上部终端分支；
2. 为真实小叶终端保留跨增强稳定候选；
3. 为shoot侧基部过渡区保留自动结构候选；
4. 短枝仅在已有学习点支持时，结合点置信度、局部叶宽和局部分支长度保留；
5. 禁止骨架自动补造没有学习点支持的叶尖；
6. 不规定固定点数，不使用人工关键点坐标。

## 首轮方案C自动结果（尚待人工复核）

- 216张train教师目标点数中位数由8增至9，每株最多一个identity视图基部覆盖角色；
- 80轮同种子训练最佳内部验证损失0.3312；
- 40张val点数中位数4，重复性F1中位数0.933；
- 组合方案输出74条候选路径，旧路线为64条；
- 已知漏叶中`v4_val_0002`、`0015`、`0039`出现路径恢复，但`0012`、`0023`、`0031`为零路径失败；
- 自动结果只能说明“值得复核”，不能说明新增路径都是真叶。

## 下一轮复核安排

1. 只使用train生成新教师并重新训练；
2. 只在val生成新路径；
3. 由两名复核者先在平台三独立完成方案C路径审核；
4. 独立审核提交后再使用平台四定位教师与解码器贡献；
5. 先计算一致性，再共同裁决分歧；
6. 建议工程门槛：严格联合通过至少32/40、漏叶不超过4张、错连不超过3张、基部wrong不超过2张、重复性F1不低于0.90；
7. 通过后才建立人工长度、角度和曲率参考；
8. 消融、五随机种子和V4 test继续后置。

该门槛用于决定是否进入表型测量，不是论文最终测试结论。
